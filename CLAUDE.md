# CLAUDE.md — Салон краси «Світлана»

Інструкції для Claude Code при роботі з цим репозиторієм.

## Про проєкт
Односторінковий сайт-візитка (лендинг) для салону краси «Світлана» у Києві.
Головна мета — щоб відвідувач міг **одразу зателефонувати та записатися на візит**.

Фронтенд: **чистий HTML + CSS + JavaScript** (без збірників, без залежностей).
Бекенд: одна **Vercel Serverless Function** (`api/callback.js`) для форми запису → Telegram.

## Структура
- `index.html` — уся розмітка сторінки (шапка, hero, послуги, про нас, ціни, контакти, футер).
- `styles.css` — стилі та адаптивність. Палітра й змінні — у блоці `:root`.
- `script.js` — рік у футері + валідація й відправка форми на `/api/callback`.
- `api/callback.js` — серверна функція: приймає POST `{name, phone, website}`, валідує,
  надсилає заявку в Telegram. Токен читається з env-змінних.
- `package.json` — `"type": "module"` (функція написана в ESM).
- `.env.example` — перелік потрібних змінних оточення.

## Форма → Telegram
Потік: браузер `POST /api/callback` → серверна функція → Telegram Bot API `sendMessage`.
Токен бота **ніколи** не потрапляє у фронтенд — лише в env-змінні на Vercel.
- Захист від спаму: приховане honeypot-поле `website` (якщо заповнене — заявка тихо ігнорується).
- Змінні оточення: `TELEGRAM_BOT_TOKEN`, `TELEGRAM_CHAT_ID`.

## Як запустити локально
Тільки фронтенд (форма покаже помилку відправки — це нормально без бекенду):
```
python -m http.server 8000
```
Разом із серверною функцією (потрібен Vercel CLI + `.env.local` зі змінними):
```
npm i -g vercel
vercel dev
```

## Правила при редагуванні
- Мова інтерфейсу — **українська**.
- Не додавати важких залежностей чи фреймворків без потреби — зберігати «нульовий білд».
- Телефон вказаний як плейсхолдер `+380000000000` у чотирьох місцях:
  шапка, hero, блок послуг (CTA), блок контактів + sticky-кнопка. **Замінювати всюди одночасно**
  (використовуй `replace_all` для рядка `tel:+380000000000`).
- Ціни в прайсі — **орієнтовні**. Реальні значення уточнює власник.
- Дизайн: елегантна пудрово-золота палітра; шрифти Cormorant Garamond (заголовки) + Montserrat (текст).
- Зберігати семантичну розмітку та доступність (aria-label на кнопках-іконках).
